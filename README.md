# Push Swap

A sorting algorithm project written in C as part of the 42 curriculum.

Push Swap focuses on algorithmic thinking, data structures, optimization, and problem-solving under strict operation constraints. The objective is to sort a stack of integers using the smallest possible number of operations with a limited set of stack manipulation instructions.

This implementation uses the Turkish Algorithm strategy for efficient chunk-based sorting and optimized move calculation.
